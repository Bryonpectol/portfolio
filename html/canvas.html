Canvas Redesign Project

This project was from my DGM 1240 class @ UVU. Each Assignment had to be completed in about a week. The whole project took about 8 weeks to complete. We had to redesign Instructure's educational system called, Canvas. Canvas is a website for students in school and is built for viewing classes, grades, assignments, and anything else the student needs during school. It uses a dashboard design to display it's content to the user. I liked the way my designs turned out, I think they are very modern looking and easy to navigate. I am currently working on prototypes for this project and will be done soon!
